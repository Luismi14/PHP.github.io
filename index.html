Crear un Usuario en MySQL y Conectarse a phpMyAdmin

Guía paso a paso para crear un usuario, asignar privilegios y conectarse a MySQL a través de phpMyAdmin.

Introducción

En esta guía aprenderás cómo crear un usuario en MySQL, configurar un host, asignar privilegios, y conectarte a tu base de datos a través de **phpMyAdmin**.

Es importante contar con acceso a la terminal de tu servidor o VPS donde tengas instalado MySQL y phpMyAdmin.

1. Acceder a MySQL

Abre la terminal y accede a MySQL como el usuario root con el siguiente comando:

mysql -u root -p

Te pedirá la contraseña del usuario root. Después de ingresarla, estarás dentro de la consola de MySQL.

2. Crear un Usuario

Para crear un nuevo usuario, ejecuta el siguiente comando en MySQL. Asegúrate de reemplazar 'nombre_usuario' y 'contraseña' por los valores que desees:

CREATE USER 'nombre_usuario'@'localhost' IDENTIFIED BY 'contraseña';

Este comando crea un usuario que solo podrá acceder a MySQL desde el mismo servidor (localhost).

Comando para crear el usuario

3. Asignar Privilegios al Usuario

Ahora debes asignar los privilegios adecuados al usuario. Para otorgarle todos los privilegios sobre una base de datos específica, usa el siguiente comando:

GRANT ALL PRIVILEGES ON nombre_base_de_datos.* TO 'nombre_usuario'@'localhost';

Si deseas que el usuario tenga privilegios globales sobre todas las bases de datos, usa:

GRANT ALL PRIVILEGES ON *.* TO 'nombre_usuario'@'localhost';

Recuerda reemplazar nombre_base_de_datos por el nombre de tu base de datos.

Comando para asignar privilegios

4. Aplicar los Cambios

Después de otorgar privilegios, asegúrate de que los cambios se apliquen con el siguiente comando:

FLUSH PRIVILEGES;

Este comando recarga los privilegios para que el nuevo usuario pueda acceder a las bases de datos con los permisos que le has otorgado.

Comando para aplicar los cambios

5. Acceder a phpMyAdmin

Una vez que el usuario esté creado y configurado, podrás acceder a **phpMyAdmin**. Ve a la página de inicio de sesión de phpMyAdmin en tu navegador y usa las credenciales del usuario creado.

En el formulario de login, usa nombre_usuario como el nombre de usuario y la contraseña que definiste. Asegúrate de que el host esté configurado como localhost.

Con esto, podrás gestionar tus bases de datos de forma gráfica y sencilla.

6. Verificación de Servicios MySQL

Para verificar si MySQL está corriendo correctamente en tu servidor, puedes usar el siguiente comando:

systemctl status mysql.service

Esto te mostrará si el servicio de MySQL está activo y funcionando.

Además, para comprobar que MySQL está escuchando en el puerto adecuado, ejecuta:

ss -tulpn | grep 3306

Esto te dirá si MySQL está escuchando en el puerto 3306 en todas las interfaces de red.
